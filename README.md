# \# 🎯 Online Quiz Application (C++)

# 

# \## 📌 Overview

# 

# The "Online Quiz Application" is a console-based project developed using "C++" that allows users to attempt multiple-choice questions and receive instant results.

# This project demonstrates core "Object-Oriented Programming (OOP)" concepts in a simple and interactive way.

# 

# \---

# 

# \## 🚀 Features

# 

# \* 🧠 Multiple-choice quiz system

# \* 👤 User input handling

# \* ✅ Answer validation

# \* 📊 Score calculation and display

# \* 🏗️ Modular OOP design (Classes \& Objects)

# \* 📐 UML diagrams included

# 

# \---

# 

# \## 🛠️ Technologies Used

# 

# \* \*\*Language:\*\* C++

# \* \*\*Concepts:\*\* OOP (Encapsulation, Classes, Objects)

# \* \*\*Tools:\*\* Any C++ Compiler (CodeBlocks / VS Code / Dev C++)
